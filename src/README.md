## Project Operational Steps

Step 1: In the terminal where the project is located, type and run:
   ```bash
   ./gradlew clean build
   ```
Step 2: In the terminal, type and run:
   ```bash
   ./gradlew bootRun
   ```
   Or just click the Run button in the top right corner of the IDE.
